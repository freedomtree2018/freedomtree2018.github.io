手把手教你用 Hexo + GitHub Pages 搭建个人博客

Posted on 2021-04-01 Edited on 2026-04-27 In 技术教程

🚀 前言

在这个信息爆炸的时代，拥有一个属于自己的独立博客显得尤为珍贵。它不仅是记录学习笔记、沉淀技术思考的最佳场所，更是打造个人技术品牌的有力武器。

市面上有很多建站方案，但如果你追求免费、稳定、高颜值且专注于写作，那么 Hexo + GitHub Pages 绝对是目前的黄金搭档。

今天，我就把自己从零开始搭建博客的全过程整理出来，希望能帮你少走弯路，快速拥有自己的“技术小窝”。

🛠️ 准备工作

在开始之前，请确保你的电脑上已经安装了以下环境（如果还没安装，可以先去官网搞定）：

  1. Node.js：Hexo 的运行环境（建议下载 LTS 长期支持版）。
  2. Git：用于代码版本控制和部署。

💡 小贴士：可以在终端输入 git --version 和 node -v 来检查是否安装成功。

从环境配置到主题美化，虽然中间可能遇到了一些报错（比如 No layout 的警告），但当看到自己的文章在互联网上被访问时，那种成就感是无与伦比的。
