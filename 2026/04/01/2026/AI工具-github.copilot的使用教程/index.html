AI工具-GitHub Copilot的使用教程

🚀 第一步：安装与激活

安装插件

在 VS Code 的扩展市场搜索 “GitHub Copilot” 并安装（选择官方插件）。

注意：

  • 如果搜索后第一个插件显示启用/禁用AI功能而非安装按钮，说明插件已安装，只需启用即可。
  • 确保安装的是官方 GitHub Copilot 插件，避免第三方仿冒。

检查安装状态

  1. 右下角状态栏： 查看是否有 Copilot 图标（类似两个气泡或 GitHub 猫头）。如果有，插件已安装。
  2. 侧边栏： 点击左侧活动栏的 Copilot 图标（对话气泡形状）。如果能打开聊天窗口，说明正常。

提示： 如果侧边栏无图标，尝试重启 VS Code 或检查扩展是否启用。

激活与登录

  • 如果右下角图标无禁止符号，已激活。
  • 如果有禁止符号，点击图标，选择“Sign in to GitHub”登录。
  • 若登录失败，访问 GitHub Copilot 官网，登录账号并激活免费计划（Copilot Free）。

常见问题： 如果点击图标出现报错，刷新 VS Code 或重新登录 GitHub。

测试激活

打开代码文件（如 .js），写注释如 // 创建一个红色的按钮，按回车查看灰色建议。若出现，按 Tab 采纳，说明激活成功。

⚙️ 开始使用

安装并激活后：

  1. 登录 GitHub 账号： 右下角提示或侧边栏登录。
  2. 确认订阅： GitHub Copilot 提供免费层级（Copilot Free），包括每月 2000 次代码补全。学生和开源维护者可免费升级。普通用户可试用后订阅（个人版约 $10/月）。
  3. 开始编码：
    • 代码补全： 写代码时，Copilot 以灰色字体建议，按 Tab 采纳。
    • AI 聊天： 点击侧边栏 Copilot 图标或按 Ctrl+L，提问或生成代码。
